# Crime Intelligence Report Using Power BI

## Project Overview
This project focuses on analyzing crime data using Power BI dashboards and data visualization techniques. The dashboard helps identify crime trends, hotspots, crime categories, and case resolution performance through interactive visualizations and DAX calculations.

## Tools & Technologies
- Power BI
- DAX
- Microsoft Excel

## Key Features
- Crime trend analysis
- Crime hotspot identification
- Resolution rate analysis
- Interactive dashboards
- Data visualization and reporting

## Learning Outcome
This project improved my skills in data analytics, dashboard development, data modeling, and visualization using Power BI.
